## 6. VARIATION SET (CONTENT POOL FOR QUESTIONS)

### Based on Place Value Differences

* Tens are different
* Tens are same and ones are different
* Tens are same, ones are same, tenths are different
* Tens → ones → tenths → hundredths → thousandths variations

---

### Based on Starting Point

* When the decimal starts from tens place
* When the decimal starts from ones place

---

### Ones-based Variations

* Ones are different
* Ones same, tenths different
* Ones same, tenths same, hundredths different
